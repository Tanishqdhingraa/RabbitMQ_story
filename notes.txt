1. docker --version
2. docker ps

3. Run RabbitMQ
docker run -d --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3-management

(If port busy)
docker run -d --name rabbitmq -p 5673:5672 -p 15673:15672 rabbitmq:3-management

4. Open UI
http://localhost:15672 (or 15673)

5. Login
guest / guest  (or admin / admin123)

6. App connection
amqp://localhost:5672 (or 5673)

7. Reset
docker stop rabbitmq
docker rm rabbitmq
